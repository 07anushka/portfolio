## рџ“¦ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/portfolio.git
```

Move into the project directory

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
